## 项目介绍
当前项目是博客 [《动态 WebApi 引擎使用教程（3行代码完成动态 WebApi 构建）》](https://www.cnblogs.com/szw/p/WebApiEngine.html) 的示例代码源码，包含了部分 WebApiEngine 功能的演示。

## 文件说明

> WebApiSample.sln：解决方案文件

> /WebApiSample：项目目录

> WeatherService.cs：演示将 Service 中的逻辑方法直接设置为 WebApi

> Startup.cs：包含了配置展示
